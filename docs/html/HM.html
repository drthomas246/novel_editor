Help Menu module¶

class HM.HelpMenuClass(app, title_binary)[ソース]¶

ベースクラス: object

ヘルプメニューバーのクラス.

・ヘルプメニューバーにあるプログラム群

パラメータ

  • app (instance) -- MainProcessingClassインスタンス

  • title_binary (str) -- タイトルイメージファイルバイナリ

help()[ソース]¶

・ウエブブラウザを使ってREADME.htmlを表示する。

version()[ソース]¶
